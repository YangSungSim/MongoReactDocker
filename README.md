# MongoReactDocker
Udemy에서 mongo와 React  멀티 컨테이너 도커 구축하기

# 실행 명령어
docker-compose up

# 유의 사항
node-js가 stable 한 버전인 16.13.0 인지 확인 후 배포
여기서 에러날시 고치고 이미지 삭제 후 다시 docker-compose up 실행
